# 嵌入式小组件

嵌入式小组件可以快速嵌入到你的项目中，实现快速开发。

仅需通过使用`iframe 标签引入` 使用一段`js 代码`调用即可。

``` js
<iframe src="https://open.youloge.com/sso#hash" width="100%" height="100%" frameborder="0"></iframe>
<script>
window.onmessage = function () {

}
</script>
```
